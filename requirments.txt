# Requirements

## Python Dependencies
- requests==2.28.2
- python-dotenv==1.0.0
- pytest==7.3.1  # For testing

## API Dependencies
- Groq Cloud API Access
- LLama 70b Model

## Installation
```bash
pip install -r requirements.txt
```

## Environment Setup
Create a `.env` file with:
```
GROQ_API_KEY=your_groq_api_key_here
```
